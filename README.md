# 信信通

信信通（xinxintong）是一个采用 MIT 协议（MIT-licensed）的开源项目。它是一个开源社群运营平台。

版本：1.0

# 部署

## 在容器中运行

> docker-compose up

## 本地化配置

新建`docker-compose.override.yml`文件。

新建`cus`目录，新建`config.php`文件和`db.php`文件，设置系统参数。

在`kcfinder`目录下，新建`upload`目录，保存上传文件。
