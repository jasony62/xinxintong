# JSONSchema

定义`schema`。

| 属性     | 类型   | 说明                                             | 必填 | 默认值                         |
| -------- | ------ | ------------------------------------------------ | ---- | ------------------------------ |
| \$schema | string | 声明该 json 是`JSONSchema定义`，而不是普通文档。 | 是   | http://json-schema.org/schema# |
| \$id     | string | 定义的唯一标识。                                 | 是   |                                |

定义单个属性（property）。

| 属性        | 说明     | 必填 | 默认值 |
| ----------- | -------- | ---- | ------ |
| type        | 属性类型 | 是   |        |
| title       | 属性标题 | 是   |        |
| description | 属性描述 | 是   |        |
| default     | 默认值   | 是   |        |
| examples    | 示例     | 是   |        |

支持的`type`取值为：`string`，`number`，`integer`，`object`，`array`，`boolean`，`null`。

每种类型支持特定属性，详细信息查看在线文档。

[Understanding JSON Schema](https://json-schema.org/understanding-json-schema/)

设置参考答案

用分号（英文半角）区分参考答案条件组，每组代表 1 项打分条件，条件组之间是或的关系，例如：`关键词1；关键词2;关键词3`。在条件组后用方括号`[]`指定需要符合的条件组数量，例如：`[2]`代表至少满足两组才给分。如果满足数量指定的是`[%]`，代表按符合条件的百分比给分。如果不指定符合数量，只要有一组满足就给分。

条件组中可用空格和逗号区分条件，条件间是和的关系，都满足才给分，例如：`关键词1,关键词2`。

`关键字1;关键字2;关键字3`，答案中包含任意 1 个就给分。

`关键字1;关键字2;关键字3[2]`，答案中包含任意 2 个给分。

`关键字1;关键字2;关键字3[%]`，答案中包含几个，按比例给分。
